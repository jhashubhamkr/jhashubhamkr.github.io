The primary objectives are to improve system scalability, data management, and overall application responsiveness.

Role and Responsibilities:

As a Backend Developer in this project, I play a major role in designing and developing RESTful APIs to support various application modules. Leveraging Spring Boot, JPA, and Hibernate ORM, I implement robust backend features and optimized data models for seamless integration with Salesforce, Oracle DB, and external REST APIs.

Technical Details:

  • API Development:
    • Designed and developed RESTful APIs to support various modules within the application.
    • Utilized Spring Boot, JPA, and Hibernate ORM for efficient backend development.
  • Microservices Architecture:
    • Implemented microservices architecture using Spring Boot-based services interacting through REST APIs and GraphQL.
    • Leveraged multi-threading techniques to optimize performance and reduce query times.
  • Data Management:
    • Managed data from various sources including Salesforce, Oracle DB, and REST APIs.
    • Developed data models using Hibernate POJOs and performed data transformations using the Map-struct library.
  • CI/CD Pipeline Improvements:
    • Contributed to enhancing CI/CD pipelines by creating a shared library and integrating with GitLab APIs, resulting in streamlined deployment processes and automation efficiency.

Achievements and Impact:

Throughout the US Bank project, my contributions have had a significant impact on system performance and project efficiency. I have successfully delivered multiple optimized GraphQL solutions. By maintaining high unit test code coverage (97%–100%), I ensured robust application quality and reliability. Additionally, I contributed to understanding new technologies through Proof of Concepts (POCs), enhancing project efficiency, and providing valuable insights for future development initiatives.

Case Study: ProBooking Appointments Management Platform

Project Overview:

As a Full Stack Developer for ProBooking (https://getprobooking.com/), I contributed significantly to the development and enhancement of an appointments management platform. The project aimed to optimize booking processes and improve user experience for a wide range of clients.

Technical Details:

In this role, I leveraged various technologies and frameworks:

  • Backend Development:
    • Developed the application using Laravel MVC framework, utilizing features like Eloquent ORM, Database Migrations, and the Blade templating engine.

  • Frontend Development:
    • Transformed the application outlook and design using Vue.js and Vuetify.js, incorporating HTML/CSS/JS for frontend enhancements.
    • Created dynamic user interfaces to enhance user engagement and optimize frontend performance.

    • Developed and managed new APIs based on existing business logic, enabling frontend consumption and enhancing overall platform functionality.

    • Integrated third-party Calendar APIs (Google Calendar, Outlook Calendar) to enable seamless scheduling and availability management.
    • Improved the backbone algorithm for handling user availabilities and multiple calendars, enhancing booking efficiency and user flexibility.

Achievements and Impact:

My contributions to ProBooking significantly improved its functionality and performance by managing the platform in a production environment, handling bug fixes and technical support for active users, and enhancing user experience and efficiency, leading to increased engagement and business growth.
